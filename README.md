# account transfer api

### Sobre:
Está API tem o objetivo de realizar transferencia de valores entre contas, a mesma foi desenvolvida utilizando as tecnologia listada abaixo:

* Java 17
* Maven
* FeignClient
* Lombok
* Junit
* Mockito
* Cache Redis
* Docker
* [Wiremock](https://wiremock.org/)
